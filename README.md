## Overview

A collection of research papers, security audits, and technical write-ups focused on blockchain security, cryptography, and software engineering. Topics include EVM smart contract analysis, attack breakdowns, zero-knowledge proofs, and performance optimization.

## Publications

### Security Research
- **[Analyzing the OpenSea Attack](./papers/opensea-attack)** - Breakdown of the 640 ETH phishing attack on OpenSea
- **[Analyzing the Ronin Attack](./papers/ronin-attack)** - Analysis of the Ronin bridge exploit

### Smart Contract Analysis
- **[Diving Into Smart Contract Decompilation](./papers/diving-into-decompilation)** - Deep dive into EVM bytecode decompilation

- **[EVM Compiler Fingerprinting](./papers/evm-compiler-fingerprinting)** - Identifying compilers from bytecode patterns
